Hey let me help you install TWRP :

TWRP Flashing Steps :



  • First Download the TWRP from here in your PC ( Download )
  • Download the boot.img file of the miui version you are using in your mobile (Soon we will provide the boot.img of miui roms)
  • Download the platform tools in your pc ( From here )
  • Now extract the platform tools on a particular location in your PC.
  • Download the adb and fastboot drivers in your PC ( here )
  • Now open the command prompt in the platform tools folder.
  • Connect the PC and your device using a data transfer cable.
  • Enable the USB Debugging option from developer option in your mobile.
  • Write command in cmd :
    1. adb reboot bootloader (Now your device will be in fastboot mode)
    2. fastboot flash boot "path of your TWRP file location"
    3. fastboot reboot recovery
  • Now go to install section in your device and open the download folder
  • Now click the install image option shown in the bottom.
  • Now click on boot image and swipe to flash.
  • After flashing boot image go to advance option in TWRP.
  • Choose option to flash current TWRP and swipe to flash twrp.
  • Now reboot to system.
  • Congratulation, TWRP flashed successfully.
